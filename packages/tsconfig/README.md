# `@rajzik/tsconfig`

Sharable typescript configuration.

## Installation

```sh
npm install --save-dev @rajzik/tsconfig typescript
yarn add --dev @rajzik/tsconfig typescript
```

## Usage

`tsconfig.json`:

```json
{
  "extends": "@rajzik/tsconfig"
}
```

Or

```json
{
  "extends": "@rajzik/tsconfig/tsconfig.workspace.json"
}
```
